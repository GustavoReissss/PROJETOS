Desenvolva com ferramentas que você já ama

Construído em código aberto

O Siteleaf é compatível com o Jekyll , o gerador de sites estáticos para mais de um milhão de sites executados no GitHub Pages.

Beneficie-se de uma grande comunidade de código aberto. Use qualquer tema Jekyll existente ou escreva o seu próprio usando Liquid e todas as ferramentas que você ama. Desenvolva localmente e até mesmo offline.

Integração com GitHub

Não há mais substituições ou exclusões acidentais. Qualquer alteração feita no Siteleaf pode ser sincronizada com o GitHub , onde você pode ver um log de edições e reverter para qualquer estado, dando a você uma máquina do tempo para seu conteúdo.

Conforme você desenvolve seu tema, o Siteleaf também pode atualizar automaticamente para refletir seus commits do GitHub. Encaixa-se perfeitamente no seu fluxo de trabalho existente.

Lançar sites melhores

Acreditamos que o gerenciamento de conteúdo não deve ser difícil. Que você deve ser capaz de acessar seus dados e hospedar seu site como quiser. Que os sites devem ser capazes de sobreviver ao seu CMS e nunca ficar presos a um fornecedor.

Nossas ferramentas devem ser simples, mas nunca simplificadas.

Sites estáticos são mais rápidos, mais seguros e mantêm os visitantes felizes. Ao contrário de CMSs legados como WordPress e Drupal, o código-fonte e o conteúdo do seu site devem ser acessíveis e facilmente movidos. Uma web aberta é uma web melhor.

Nos últimos anos, geradores de sites estáticos ganharam popularidade. Mas junto com isso veio muito inchaço (e novas siglas). Vamos manter os sites estáticos como eles foram concebidos: rápidos e simples.
